ðŸ”¹ Project Structure
The project consists of the following main components:

A src/test/java directory that contains:

A pages package where all Page Object Model classes are implemented.

A tests package that includes the actual test scripts.

A utils package used for helper classes such as data readers, configuration loaders, and reusable methods.

A src/test/resources directory that contains:

A JSON file that holds the test data used for data-driven testing.

A properties file for configuration values such as URLs and credentials.

A pom.xml file that manages all project dependencies and build configuration.

A README.md file that documents the project and explains how it works.
